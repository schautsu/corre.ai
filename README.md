# CORRE.AI

> Este é um jogo no estilo pega-pega, onde um jogador corre atrás de outro. O jogador que conseguir se manter longe do oponente ou capturá-lo dentro do tempo determinado vence.

- **Disponível para**: Windows
- **Versão**: Unity 6000.0
